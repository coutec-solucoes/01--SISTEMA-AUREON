# Console Visual de Homologação (Blazor)

Para unificar o uso de todos os artefatos de diagnóstico, o sistema Aureon dispõe agora de um "Console" (Fase 19, Blocos 7 e 9).

## Características
- **Acesso UI**: `/fiscal/homologacao-console` e `/fiscal/prontidao-homologacao`.
- Mostra a disponibilidade das conexões.
- Lista as configurações de servidor das APIs fiscais.
- Acompanha o Checklist (Prontidão Técnica).
- É a UI onde ficam expressamente detalhados os avisos: **"NÃO TEM VALIDADE JURÍDICA"**, indicando que trata-se unicamente de telemetria da arquitetura, sem permissão para botões de "Emitir" reais até a Fase 20.
